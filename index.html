How the Model Works

  • 1. Data Collection: Gathers data such as grades, attendance, study hours, and social media usage from students.
  • 2. Data Preprocessing: Cleans the data and uses techniques like feature encoding to prepare it for analysis.
  • 3. Model Selection: Uses Random Forest for classification and SMOTE to handle class imbalance for higher accuracy.
  • 4. Prediction: Inputs are passed through the trained model to predict whether a student is likely to pass or fail.
  • 5. Personalized Output: Recommendations are provided including weak subjects, YouTube tutorial links, and tips to improve study habits and well-being.
  • 6. Real-Time Access: Deployed on Streamlit so users can interactively test the model from any device.
